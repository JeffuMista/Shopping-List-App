A responsive mobile App created using Javascript, HTML, and CSS
App makes it seamless to shop by helping users organize items and check them off as they shop.

Features
Allows you to add items via input field by pressing enter or clicking on button
Save the shopping list items to a real-time Firebase database
Check-off the items while shopping - this removes them from the shopping list items and the database
No special set-ups/installations needed to test the app.
